可以看到进队管理员权限进行校验，而没有对提交token进行校验，导致恶意用户可引导管理员点击构造的url进行管理员添加，实际操作可不跳转到管理页面，以免引起怀疑。

漏洞复现

登陆后台，在高级选项>管理员>新增管理员点击之后截包
